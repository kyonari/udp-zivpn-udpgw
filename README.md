>Server binary for Linux amd64 and arm.

#### Installation
```
wget -O zi.sh https://raw.githubusercontent.com/kyonari/udp-zivpn-kyo/main/zi.sh; sudo chmod +x zi.sh; sudo ./zi.sh
```
